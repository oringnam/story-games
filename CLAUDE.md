# CLAUDE.md - Story Games

분기형 스토리 게임 제작 프로젝트.

## 프로젝트 개요

**목표**: 모바일에서 실행 가능한 선택지 기반 인터랙티브 스토리 게임  
**현재 상태**: 초기 세팅 (2026-02-14)  
**레포**: story-games (로컬 → GitHub 예정)

## 핵심 원칙

1. **모바일 우선**: 터치 최적화, 세로 화면
2. **간결함**: Vanilla JS, 프레임워크 없음
3. **데이터 기반**: JSON으로 스토리 관리
4. **재사용성**: 공통 모듈 활용

## 구조

### games/
개별 스토리 게임들. 각 게임은 독립적.

### common/
모든 게임이 공유하는 모듈:
- **story-engine.js**: 씬 렌더링, 분기 처리, 히스토리 관리
- **save-system.js**: localStorage 기반 세이브/로드
- **ui-components.js**: 공통 UI (버튼, 모달, 메뉴 등)

### assets/
공통 에셋 (사운드, 이미지)

## 스토리 데이터 구조 (초안)

```json
{
  "title": "숲의 선택",
  "scenes": {
    "start": {
      "text": "당신은 어두운 숲에 서 있습니다.",
      "choices": [
        { "text": "왼쪽 길로 간다", "next": "left_path" },
        { "text": "오른쪽 길로 간다", "next": "right_path" }
      ]
    },
    "left_path": {
      "text": "...",
      "choices": [...]
    }
  }
}
```

## 개발 워크플로우

1. 로컬에서 작업
2. `python3 -m http.server 8000`으로 테스트
3. 모바일 (폰/태블릿)에서 확인
4. Git commit
5. GitHub Pages 배포
